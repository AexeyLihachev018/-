# Процесс: Ежедневный отчёт РОПа — зелёная таблица

> Таблица: «Отчёт РОПа» (Google Sheets / Excel)
> Цель: ежедневный учёт результатов каждого менеджера по продажам
> Аудитория: Директор + РОП

---

## 1. ЧТО ЗАПУСКАЕТ ПРОЦЕСС? (триггер)

**Основной триггер:** Конец рабочего дня (17:00–18:00)

РОП собирает данные за прошедший день по каждому менеджеру:
- Открывает Битрикс24 → смотрит сделки и лиды за день
- Считает звонки по каждому менеджеру из телефонии
- Считает отправленные КП из почты / Б24
- Считает новые лиды, закрытые сделки, сумму

**Периодичность:** Каждый рабочий день
**Кто запускает:** РОП (руководитель отдела продаж)

---

## 2. ПОШАГОВЫЕ ДЕЙСТВИЯ

### Шаг 1: Сбор данных из Битрикс24
| Кто | Действие | Откуда | Время |
|---|---|---|---|
| РОП | Открывает Б24 → лиды за сегодня | Б24 CRM | 5–10 мин |
| РОП | Считает новые лиды по каждому менеджеру | Б24 CRM | 5 мин |
| РОП | Считает сделки на каждой стадии | Б24 CRM | 5–10 мин |
| РОП | Смотрит сумму закрытых сделок | Б24 CRM | 5 мин |

### Шаг 2: Сбор данных из телефонии
| Кто | Действие | Откуда | Время |
|---|---|---|---|
| РОП | Открывает кабинет телефонии | Mango / Телефон.ру | 5 мин |
| РОП | Считает исходящие/входящие звонки по каждому менеджеру | Телефония | 5–10 мин |

### Шаг 3: Ручной ввод в зелёную таблицу
| Кто | Действие | Время |
|---|---|---|
| РОП | Открывает Google Sheets → вкладка текущего месяца | 2 мин |
| РОП | Находит строку сегодняшней даты | 2 мин |
| РОП | Вносит данные по каждому менеджеру (5 менеджеров × 7 полей = 35 ячеек) | 15–20 мин |
| РОП | Проверяет что оранжевая таблица обновилась корректно | 5 мин |
| РОП | Исправляет ошибки #DIV/0! если появились | 3–5 мин |

**Итого на весь процесс: 45–70 минут в день**

---

## 3. ЧТО НА ВЫХОДЕ?

| Результат | Формат | Получатель |
|---|---|---|
| Зелёная таблица заполнена | Google Sheets | РОП |
| Оранжевая сводка обновлена | Google Sheets (формулы) | Директор, РОП |
| Картина по каждому менеджеру | Цифры в таблице | Директор |
| Понимание: кто выполняет план | Визуальный анализ | Директор, РОП |

**Ключевые данные в отчёте:**
- Количество новых лидов по менеджеру
- Количество звонков (входящих / исходящих)
- Количество отправленных КП
- Количество закрытых сделок
- Выручка за день
- Конверсия лид → сделка
- % выполнения месячного плана

---

## 4. ГДЕ МОЖНО СЛОМАТЬСЯ? (типичные ошибки)

| # | Проблема | Последствие | Частота |
|---|---|---|---|
| 1 | РОП заболел / в командировке | Данные не вносятся 1–3 дня | Редко, но критично |
| 2 | Ошибка #DIV/0! в оранжевой таблице | Директор видит некорректные данные | Часто |
| 3 | Данные из Б24 расходятся с телефонией | Непонятно какой цифре верить | Часто |
| 4 | Менеджер закрыл сделку задним числом | Данные за прошлые дни некорректны | Средне |
| 5 | Перепутали вкладку месяца | Данные внесены не туда | Редко |
| 6 | РОП вносит данные «на глаз» без проверки Б24 | Приписки, ошибки | Средне |

## 5. ПРЕДЛОЖЕНИЕ ПО АВТОМАТИЗАЦИИ

### Цель
Полностью убрать ручной ввод данных РОПом.
Зелёная таблица заполняется сама из Битрикс24.
**РОП тратит 0 минут вместо 45–70 минут в день.**

---

### Вариант А: Быстрый — экспорт из Б24 вручную (1 день)

```
Б24 CRM → Экспорт в CSV → Google Sheets (вставить)

Как работает:
1. РОП нажимает «Экспорт» в Б24 раз в день
2. Скачивает CSV
3. Вставляет в нужную вкладку

Экономия: 30–40 мин/день (убирает ручной пересчёт)
Что остаётся вручную: сам экспорт и вставка
```

---

### Вариант Б: Оптимальный — Albato/Make: Б24 → Google Sheets (1–2 недели)

```
[Битрикс24 CRM]
      │  каждую ночь в 23:59
      ▼
[Albato / Make (Integromat)]
      │  забирает сделки, лиды, звонки за день
      ▼
[Google Sheets — зелёная таблица]
      │  автоматически заполняет нужные ячейки
      ▼
[Оранжевая таблица]
      │  формулы пересчитываются сами
      ▼
Директор видит актуальный отчёт утром без участия РОПа
```

**Что автоматизируется:** Весь сбор и ввод данных
**Что остаётся:** Только интерпретация и управленческие решения
**Инструменты:** Albato (от 990 руб/мес) или Make (от 9$/мес)
**Стоимость:** ~1 000–1 500 руб/мес

---

### Вариант В: Максимальный — дашборд прямо в Б24 (настройка 2–3 часа)

```
Все данные уже есть в Б24 →
      │
      ▼
CRM Аналитика Б24 (встроенный инструмент):
  - Воронка по менеджерам
  - Конверсии по стадиям
  - Выручка план/факт
  - Количество звонков и КП
      │
      ▼
Директор открывает Б24 → видит всё в реальном времени
Никаких таблиц, никакого ручного ввода
```

**Стоимость:** 0 руб (уже включено в Б24)
**Время настройки:** 2–3 часа

---

### Рекомендуемый план внедрения

```
СЕГОДНЯ (30 минут):
  → Открыть Б24 → CRM → Аналитика
  → Настроить отчёт по менеджерам за день
  → Добавить виджет на главную страницу директора
  → РОП сразу перестаёт вручную считать данные из Б24

НЕДЕЛЯ 1–2:
  → Подключить Albato: Б24 → Google Sheets
  → Зелёная таблица заполняется автоматически каждую ночь
  → РОП утром только проверяет, не заполняет

МЕСЯЦ 1:
  → Исправить формулы #DIV/0! в оранжевой таблице
  → Добавить сводную вкладку «Год»
  → Настроить еженедельный авторепорт директору на email
```

---

### Следующий шаг: включить CRM Аналитику в Б24

Это бесплатно и занимает 30 минут:
1. Открыть Б24 → CRM → Аналитика (левое меню)
2. Выбрать «Отчёт по сотрудникам»
3. Настроить нужные поля и период
4. Добавить на главную страницу директора через «Виджеты»

**Это закрывает 80% потребности без единой интеграции.**
